## üé® Quick Screenshot Tips

### For Project Screenshots:

1. **Use Full HD Resolution**

   - Capture at 1920x1080 or higher
   - Resize to 1200x630 for consistency

2. **Clean the Interface**

   - Close unnecessary tabs
   - Hide personal information
   - Use demo data if needed

3. **Highlight Key Features**

   - Show the main interface
   - Include important metrics
   - Display working features

4. **Good Lighting/Theme**
   - Use dark mode for modern look
   - Ensure text is readable
   - Avoid glare or reflections

### Tools You Can Use:

- **Windows:** Snipping Tool, Snip & Sketch (Win + Shift + S)
- **Browser:** DevTools responsive design mode
- **Screenshot Tools:** Lightshot, ShareX, Greenshot
- **Image Editing:** Canva, Photoshop, GIMP, Paint.NET
